hans.geometry.fourier_synthesis(shape, size, Hurst, rms_height=None, rms_slope=None, short_cutoff=None, long_cutoff=None, rolloff=1.0)ÔÉÅ

Create a self-affine, randomly rough surface using a Fourier filtering algorithm. Adapted from https://github.com/ContactEngineering/SurfaceTopography. The algorithm is described in: Ramisetti et al., J. Phys.: Condens. Matter 23, 215004 (2011); Jacobs, Junge, Pastewka, Surf. Topgogr.: Metrol. Prop. 5, 013001 (2017)

Parameters:

  • shape (tuple of ints) ‚Äì Number of grid cells in x and y direction

  • size (tuple of floats) ‚Äì Physical size in x and y direction

  • Hurst (float) ‚Äì Hurst exponent.

  • rms_height (float) ‚Äì Root mean-squared height.

  • rms_slope (float) ‚Äì Root mean-squared slope.

  • short_cutoff (float) ‚Äì Short-wavelength cutoff.

  • long_cutoff (float) ‚Äì Long-wavelength cutoff.

  • rolloff (float) ‚Äì Value for the power-spectral density (PSD) below the long-wavelength cutoff. This multiplies the value at the cutoff, i.e. unit will give a PSD that is flat below the cutoff, zero will give a PSD that is vanishes below cutoff. (Default: 1.0)

Returns:

Rough topography.

Return type:

np.array

hans.geometry.self_affine_prefactor(grid_pts, physical_sizes, Hurst, rms_height=None, rms_slope=None, short_cutoff=None, long_cutoff=None)ÔÉÅ

Compute prefactor \(C_0\) for the power-spectrum density of an ideal self-affine topography given by

\[C(q) = C_0 q^{-2-2H}\]

for two-dimensional topography maps and

\[C(q) = C_0 q^{-1-2H}\]

for one-dimensional line scans. Here \(H\) is the Hurst exponent.

Note: In the 2D case:

\[h^2_{rms} = \frac{1}{2 \pi} \int_{0}^{\infty} q C^{iso}(q) dq\]

whereas in the 1D case:

\[h^2_{rms} = \frac{1}{\pi} \int_{0}^{\infty} C^{1D}(q) dq\]

See Equations (1) and (4) in [1].

Parameters:

  • grid_pts (array_like) ‚Äì Resolution of the topography map or the line scan.

  • physical_sizes (array_like) ‚Äì Physical physical_sizes of the topography map or the line scan.

  • Hurst (float) ‚Äì Hurst exponent.

  • rms_height (float) ‚Äì Root mean-squared height.

  • rms_slope (float) ‚Äì Root mean-squared slope of the topography map or the line scan.

  • short_cutoff (float) ‚Äì Short-wavelength cutoff.

  • long_cutoff (float) ‚Äì Long-wavelength cutoff.

Returns:

prefactor ‚Äì Prefactor \(\sqrt{C_0}\)

Return type:

float

References

[1]: Jacobs, Junge, Pastewka, Surf. Topgogr.: Metrol. Prop. 5, 013001 (2017)
